Detailed Description

Handles logic for object detection and movement calculations. details The Controller class implements the main control loop. It arbitrates between manual joystick input and autonomous detection tracking, ensuring that the BaseMovementManager receives the correct vector for the turret motors.

void Controller::run ( )

Main execution loop for the turret system.

When called, this method polls the detection module for targets and the joystick for user input. It calculates the necessary PWM or step values and updates the movement manager state.

Used Null since it's' currently isn't attached to the camera logic
